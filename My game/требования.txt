Windows 7 (SP1+), Windows 10 и Windows 11.
Архитектура x86, x64 с поддержкой набора инструкций SSE2.
Поддержка DX10, DX11, DX12.
Поставщик оборудования официально поддерживает драйверы.
Для разработки: серверной части сценариев IL2CPP требуется Visual Studio 2015 с компонентом C++ Tools или более поздней версии и Windows 10+ SDK.
